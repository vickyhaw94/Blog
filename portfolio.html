StickFighter

"Stick fighter" is a new invented game and it's inspired by "Street fighter" and "Guess Who" from two different categories which first is arcade game and second is board game. The game required two player in order to start the game. Each round player can select either attack or defend then only select the position. This game is full of intense which player need to guess what are the opponents thinking then only counter their action. The default time limit is set to 25 second. The default health points is set to 3. The one with higher health points is the final winner.
